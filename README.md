# Ultra Hard Coremod #

## About ##
This mod is a UHC mod inspired by Gerrybrano's [UHC Blox](https://www.planetminecraft.com/project/minecraft-uhc-box/). I'm planning on adding as much support as needed to make modded UHC's possible.
